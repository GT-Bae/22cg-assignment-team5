# 2팀 과제 계획

## 目標
- 複数のゲームを一つにまとめ、短時間で気軽に楽しめる作品を制作する。

## 計画
- チームメンバー5名が、それぞれ1本ずつ短いミニゲームを制作する。
- タイトル画面は3名で分担して制作する。
- ステージ選択画面は1名が制作する。

### 詳細計画  
#### タイトル画面
```
タイトル画面は、一般的なノートパソコンの画像を用いて実装し、
以下の3つの要素で構成する。

1. 画面上にタイトルと開始ボタンを表示する。
2. マウスカーソルがノートパソコンに近づくと、
   ノートパソコンが徐々に開く演出を表示する。
   （カーソルがノートパソコンの範囲内にある間は、開いた状態を維持する）
3. ノートパソコンの周囲に、マウス操作を促す矢印を表示する。
```
##### 上記3要素をすべて表現したイメージ
![image](https://user-images.githubusercontent.com/81298756/170483842-7e2ef56f-07c5-40e7-894f-fa116c00db90.png)  
#### ゲーム画面
```
ゲーム画面は、ステージ選択画面と各ミニゲームで構成する。
```
##### ステージ選択画面のイメージ
```
開始時はステージ1のみを選択可能とする。
ステージ1をクリアするとステージ2が解放され、
ステージ2をクリアするとステージ3が解放される形式とする。
```
![image](https://user-images.githubusercontent.com/81298756/170484042-ef028aae-df42-43b6-a212-5462ee9cc121.png)  
##### ミニゲーム 
> ミニゲームは各メンバーが個別に制作するため、ゲームごとにデザインや内容が異なる予定です。
